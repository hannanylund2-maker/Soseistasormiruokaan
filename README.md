# Soseista Sormiruokaan

Sormiruoka-reseptisivusto suomalaisille perheille.

## Tiedostorakenne

```
soseistasormiruokaan/
├── public/
│   ├── index.html          ← Pääsivu
│   ├── images/             ← Ladatut kuvat
│   └── admin/
│       ├── index.html      ← CMS-hallintapaneeli
│       └── config.yml      ← CMS-asetukset
├── content/
│   └── recipes/            ← Reseptitiedostot (.json)
└── netlify.toml            ← Netlify-asetukset
```

## Uuden reseptin lisääminen

Mene osoitteeseen: `https://sinununsivustosi.netlify.app/admin`

## Kehitys paikallisesti

Avaa `public/index.html` selaimessa.
